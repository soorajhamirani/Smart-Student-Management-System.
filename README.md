# Smart Student Management System
A simple Python project made in Jupyter Notebook that manages student records using file handling.

## Features
- Add, view, search, update, delete students
- Analyze data (average, top, pass rate)
- Beginner-friendly, written only using Python functions

## How to Run
1. Open the notebook in Jupyter
2. Run all cells in order
3. The file `students.txt` will store all records
